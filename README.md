# Snake - React 

Jeu du Snake développé avec **React** dans le but d'approfondir mes connaissance en React et en gestion d'état.

## Objectifs du projet 

- Comprendre et maîtriser : 
    - les hooks React
    - la gestion du state complexe
    - les événements clavier
    - les timers ('setInterval')
- Structurer une application React proprement 
- Implémenter une logique de jeu en JavaScript
- Styliser une application avec Tailwind CSS v3

## Technologies utilisées

- React
- Vite
- Javascript
- CSS / Tailwind CSS

## Ce que j’ai appris

- Mise en place d’une boucle de jeu avec useEffect
- Manipulation d’état complexe
- Calcul de collisions
- Structuration d’une application React

## Installation

npm install
npm run dev
